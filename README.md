# 🤟 ASL Alphabet Recognition using CNN

A deep learning project that recognizes American Sign Language alphabets using a Convolutional Neural Network (CNN).

## Features

- CNN built from scratch using TensorFlow/Keras
- 25 ASL alphabet classes
- Streamlit web application
- Upload hand sign image for prediction
- Test Accuracy: 93.43%

## Technologies Used

- Python
- TensorFlow
- Keras
- Streamlit
- Pandas
- NumPy
- Matplotlib

## Files

- train_model.ipynb → Model training notebook
- asl_alphabet_model.keras → Trained CNN model
- app.py → Streamlit web app
